This project provides a low-\/level Open\+GL abstraction implemented in modern C++, using try-\/catch based error handling and classes representing various Open\+GL objects.\hypertarget{md_inc_2_e_g_l_2mainpage_autotoc_md1}{}\doxysection{\texorpdfstring{Recommended Setup}{Recommended Setup}}\label{md_inc_2_e_g_l_2mainpage_autotoc_md1}
The recommended setup is to have your main code class inherit from {\ttfamily \doxylink{classegl_1_1_window_context}{egl::\+\+Window\+Context}} and implement your logic in the provided virtual run function.

Calling {\ttfamily \doxylink{classegl_1_1_window_context_a53a5f0a67b1b943cc3de2d8f420ce45b}{egl::\+\+Window\+Context::\+use\+Context}} is recommended to avoid potential errors.

Inside of {\ttfamily \doxylink{classegl_1_1_window_context_acd76c03d3bdf79bcdd83e022cdfa1127}{egl::\+\+Window\+Context::\+run}} any Easy Open\+GL functionality can be used safely.
