\doxysubsection{Detailed Description}
\doxylink{classgla_1_1_program}{Program} class to abstract Open\+GL \doxylink{classgla_1_1_shader}{Shader} Programs. 

\begin{DoxyWarning}{Warning}
\doxylink{classgla_1_1_program}{Program} must be deconstructed before the Open\+GL context is destroyed. 

This class is not guaranteed to be thread-\/safe.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
This class owns the underlying Open\+GL \doxylink{classgla_1_1_program}{Program} object and releases it upon destruction or \doxylink{classgla_1_1_program_abab59908a135680e931c00ffbfd711c1}{reset()}. 
\end{DoxyNote}
