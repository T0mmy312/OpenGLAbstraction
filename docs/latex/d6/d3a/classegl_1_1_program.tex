\doxysubsection{Detailed Description}
\doxylink{classegl_1_1_program}{Program} class to abstract Open\+GL \doxylink{classegl_1_1_shader}{Shader} Programs. 

\begin{DoxyWarning}{Warning}
\doxylink{classegl_1_1_program}{Program} must be deconstructed before the Open\+GL context is destroyed. 

This class is not guaranteed to be thread-\/safe.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
This class owns the underlying Open\+GL \doxylink{classegl_1_1_program}{Program} object and releases it upon destruction or \doxylink{classegl_1_1_program_a0d9bf7750add025ffc566b823e6f76ad}{reset()}. 
\end{DoxyNote}
