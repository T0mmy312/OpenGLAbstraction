\doxysubsection{Detailed Description}
Virtual methods to override for window events.

\begin{DoxyWarning}{Warning}
These are callbacks, so their execution time is not guaranteed. Modifying global state within them (e.\+g., calling use\+Context or binding a \doxylink{classgla_1_1_buffer}{Buffer}) may cause undefined behavior. Use a flag to control or defer such changes. 
\end{DoxyWarning}
