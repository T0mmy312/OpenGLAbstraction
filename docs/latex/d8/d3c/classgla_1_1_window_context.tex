\doxysubsection{Detailed Description}
An abstract class to contain an window based application. 

Instantiates a window in the constructor using GLFW and destroys it once it goes out of scope. That window can be bound and used in the child class in the \doxylink{classgla_1_1_window_context_abfba43a1a6f8eb445ac2c9dd7deaf0f8}{Window\+Context::\+run} method implementation containing the setup and rendering loop.

\begin{DoxyNote}{Note}
Calling use\+Context is recomended in run to ensure the local context is used. 

\doxylink{classgla_1_1_window_context}{gla::\+\+Window\+Context} is not thread safe and may only be used on the main thread. 
\end{DoxyNote}
