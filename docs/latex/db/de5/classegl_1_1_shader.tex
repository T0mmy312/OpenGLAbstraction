\doxysubsection{Detailed Description}
\doxylink{classegl_1_1_shader}{Shader} class to abstract Open\+GL Shaders. 

\begin{DoxyWarning}{Warning}
\doxylink{classegl_1_1_shader}{Shader} must be deconstructed before the Open\+GL context is destroyed. 

This class is not guaranteed to be thread-\/safe.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
This class owns the underlying Open\+GL \doxylink{classegl_1_1_shader}{Shader} object and releases it upon destruction or \doxylink{classegl_1_1_shader_a2dda7169e2fef7b36a890887a807c18f}{reset()}. 
\end{DoxyNote}
