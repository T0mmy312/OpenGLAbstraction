\doxysubsection{Detailed Description}
An abstract class to contain an window based application. 

Instantiates a window in the constructor using GLFW and destroys it once it goes out of scope. That window can be bound and used in the child class in the \doxylink{classegl_1_1_window_context_acd76c03d3bdf79bcdd83e022cdfa1127}{Window\+Context::\+run} method implementation containing the setup and rendering loop.

\begin{DoxyNote}{Note}
Calling use\+Context is recomended in run to ensure the local context is used. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
\doxylink{classegl_1_1_window_context}{Window\+Context} may not be shared between threads, but multiple can be instantiated and used on diffrent threads as long as the GLFW Window is not shared. 
\end{DoxyWarning}
