\doxysubsection{Detailed Description}
\doxylink{classgla_1_1_shader}{Shader} class to abstract Open\+GL Shaders. 

\begin{DoxyWarning}{Warning}
\doxylink{classgla_1_1_shader}{Shader} must be deconstructed before the Open\+GL context is destroyed. 

This class is not guaranteed to be thread-\/safe.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
This class owns the underlying Open\+GL \doxylink{classgla_1_1_shader}{Shader} object and releases it upon destruction or \doxylink{classgla_1_1_shader_ac0adaac6e0537834a471049bb86c30e5}{reset()}. 
\end{DoxyNote}
